# Login with JWTs

> Using Express and MongoDb to create a login API. Users can register for an account then only add and view post when logged in. 

### Packages used
- bcrypt
- cookie-parser
- express
- jsonwebtoken
- moment
- mongoose
- nodemon
- uuid
